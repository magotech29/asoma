スーパー管理者マニュアル

ぐるっとスタンプラリー スーパー管理者画面操作ガイド

最終更新：2026年5月

1. スーパー管理者とは

スーパー管理者は、システム全体を統括する最上位の権限です。

権限 スーパー管理者 コミュニティ管理者 参加者
コミュニティの作成・編集 ✓ — —
管理者URLの発行 ✓ — —
参加者URLの発行 ✓ — —
コミュニティ管理画面への直接入場 ✓ — —
イベント・コース・スポット管理 ✓（代理） ✓ —
統計・応募者確認 ✓（代理） ✓ —
QRコード印刷 ✓（代理） ✓ —
スタンプ収集 — — ✓

2. ログイン

スーパー管理者専用のURLにアクセスします。

https://asoma.mago-t.com/super-admin/login

メールアドレスとパスワードを入力してログインします。
（認証情報はサーバー環境変数 SUPER_ADMIN_EMAIL / SUPER_ADMIN_PASSWORD で管理されています）

ログインセッションは 8時間 で切れます。再度ログインしてください。

4. コミュニティの作成

新しいコミュニティ（テナント）を作成します。

  1. 画面上部の「新しいコミュニティを追加」フォームに入力
    • コミュニティ名（必須）：例）春日市ウォーキングラリー2026
    • 管理者パスワード（必須）：このコミュニティの管理画面ログインに使うパスワード（8文字以上推奨）
  2. 「コミュニティを作成」ボタンをタップ
  3. コミュニティ一覧に追加されれば成功

作成と同時に 8文字のアクセストークン が自動生成されます。
このトークンが参加者URL・管理者URLに使われます。

5. URLの発行・コピー

各コミュニティに対して2種類のURLが自動発行されます。

参加者URL

スタンプラリーに参加する一般ユーザーが使うURL。

https://asoma.mago-t.com/?t=XXXXXXXX
  • コミュニティ一覧の「📋 参加者URL」ボタンをタップ → クリップボードにコピー
  • チラシ・SNS・回覧板などで参加者に配布する

管理者URL

コミュニティ管理者がログインするためのURL。

https://asoma.mago-t.com/admin/login?t=XXXXXXXX
  • コミュニティ一覧の「📋 管理者URL」ボタンをタップ → クリップボードにコピー
  • 担当のコミュニティ管理者にこのURLとパスワードを伝える

重要： URLに含まれる t=XXXXXXXX がそのコミュニティの識別子です。
参加者・管理者ともに、このURLから入らないとアクセスできません。

6. コミュニティの編集

既存コミュニティの情報を変更できます。

  1. コミュニティ一覧の該当コミュニティ右上の「編集」ボタンをタップ
  2. 変更したい項目を入力
    • コミュニティ名：名称を変更
    • 新しいパスワード：変更しない場合は空欄のままでOK
  3. 「保存」ボタンをタップ

7. コミュニティ管理画面への入場（代理操作）

スーパー管理者は、任意のコミュニティの管理画面に直接入場できます。
コミュニティ管理者と同等のすべての操作が可能です。

  1. 対象コミュニティの「🔑 管理画面を開く」ボタンをタップ
  2. そのコミュニティの管理画面（/admin）に自動遷移

管理画面の左上に管理中のコミュニティ名が 緑のバナー で表示されます。
例）管理中：春日市スタンプラリー

スーパー管理者画面に戻る

管理画面ヘッダー右上の「← スーパー管理者」リンクをタップすると、
スーパー管理者画面に戻れます。

8. 代理操作でできること

コミュニティ管理画面に入場した状態で、以下のすべての操作が可能です。

操作 内容
📊 統計 参加者数・完走率・スポット訪問数の確認
🎁 景品応募者 応募者一覧・当選/落選の管理
🗺️ コース管理 コースの追加・編集・削除
📍 スポット管理 チェックポイントの追加・編集・削除
🔲 QRコード QRコードの表示・印刷
⚙️ イベント管理 開催期間・コース構成の管理・コピー

詳細な操作手順は コミュニティ管理者マニュアル を参照してください。

9. 運営の流れ（初回セットアップ）

① コミュニティを作成（スーパー管理者画面）
      ↓
② 管理者URLとパスワードをコミュニティ管理者に連絡
      ↓
③ 参加者URLを配布素材に使えるよう連絡
      ↓
④ 必要に応じて「管理画面を開く」で代理セットアップ
      ↓
⑤ コミュニティ管理者がイベント・コース・スポット・QRを設定
      ↓
⑥ 開催スタート

10. よくある質問

Q: コミュニティを削除したい
A: 現在、削除機能はありません。不要なコミュニティは無効化（将来予定）するか、スーパー管理者に連絡してください。

Q: 参加者URLを変えたい
A: URLに含まれるトークンは変更できません。新しいコミュニティを作成し、URLを再配布する必要があります。

Q: パスワードを忘れた（スーパー管理者自身のパスワード）
A: サーバーの環境変数 SUPER_ADMIN_PASSWORD を変更して再起動してください。

Q: コミュニティ管理者がパスワードを忘れた
A: スーパー管理者画面から当該コミュニティを「編集」し、新しいパスワードを設定してください。

Q: 複数のコミュニティを同時に管理できるか
A: 画面上は1つずつ入場する形になります。ブラウザのタブを分けて複数管理することは可能です。
