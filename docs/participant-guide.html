ぐるっとスタンプラリー の使い方

最終更新：2026年5月

はじめに

このスタンプラリーはアプリのインストール不要で、スマートフォンのブラウザだけで参加できます。

必要なもの： スマートフォン（カメラ付き）

参加方法

ステップ1：参加者URLにアクセス

主催者から受け取ったURLをブラウザで開いてください。

https://asoma.mago-t.com/?t=XXXXXXXX

重要： このURLを最初に開くことで参加登録されます。
https://asoma.mago-t.com に直接アクセスしても参加できません。

ステップ2：コース一覧を確認する

トップページにコース一覧と各スポット（チェックポイント）が表示されます。
距離・所要時間を参考にコースを選んで出発しましょう。

ステップ3：スポットでQRコードを読み取る

  1. 「📷 QRコードをスキャン」ボタンをタップ
  2. カメラへのアクセス許可を求められたら「許可」をタップ
  3. スポットに設置されたQRコードにカメラを向ける
  4. 自動で読み取られ、スタンプが押されます 🎉

同じスポットのQRコードを何度読み取っても、スタンプは1回しか取得できません。

ステップ4：スタンプ帳を確認する

「🎯 スタンプ帳を見る」をタップすると、取得済みのスタンプと進捗が確認できます。

ステップ5：全スポット制覇で景品に応募

全スポットを集めると「🎁 景品に応募する」ボタンが表示されます。

  1. 「景品に応募する」をタップ
  2. お名前と連絡先（メールアドレスまたは電話番号）を入力
  3. 「応募する」をタップ
  4. 「応募完了！」画面が表示されたら完了

抽選結果は入力した連絡先にお知らせします。

こんなときは

カメラが起動しない

iPhone（Safari）： 設定アプリ → Safari → カメラ → 「許可」

Android（Chrome）： アドレスバー横の🔒アイコン → カメラ → 「許可」

QRコードが読み取れない

  • カメラをQRコードから20〜30cm離してみてください
  • 明るい場所で試してください
  • カメラのレンズが汚れていないか確認してください

スタンプ帳が表示されない

ページを再読み込みしてください。それでも解決しない場合は、主催者から受け取ったURLを再度開いてください（スタンプは保存されています）。

スマホやブラウザを変えた

スタンプの記録はブラウザのデータに保存されています。
スマホやブラウザを変えると以前のスタンプが引き継がれません。
同じスマホ・同じブラウザでご利用ください。

プライバシーについて

  • 氏名・連絡先は景品応募時のみ取得します
  • スタンプ記録はブラウザのCookieで管理されます
  • 個人を特定する情報は景品応募前には収集しません

ご不明な点は会場スタッフまでお声がけください
