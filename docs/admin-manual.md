# コミュニティ管理者マニュアル
## ぐるっとスタンプラリー 管理画面操作ガイド

最終更新：2026年5月

---

## 1. ログイン

スーパー管理者から発行された **管理者URL** をブラウザで開いてください。

```
https://asoma.mago-t.com/admin/login?t=XXXXXXXX
```

パスワードを入力してログインします（メールアドレスは不要）。

> ログインセッションは **8時間** で切れます。再度URLを開いてログインしてください。

---

## 2. 管理画面の構成

ログイン後、以下のメニューが表示されます。

| メニュー | 用途 |
|---------|------|
| 📊 統計 | 参加者数・完走率・スポット訪問数の確認 |
| 🎁 景品応募者 | 応募者一覧・当選/落選の管理 |
| 🗺️ コース管理 | ウォーキングコースの追加・編集・削除 |
| 📍 スポット管理 | チェックポイントの追加・編集・削除 |
| 🔲 QRコード | QRコードの表示・印刷 |
| ⚙️ イベント管理 | 開催期間・コース構成の管理 |

---

## 3. 運営の流れ（初回セットアップ）

スタンプラリーを始めるには、以下の順番で設定します。

```
① イベントを作成（⚙️ イベント管理）
      ↓
② コースを追加（🗺️ コース管理）
      ↓
③ スポットを追加（📍 スポット管理）
      ↓
④ QRコードを印刷して設置（🔲 QRコード）
      ↓
⑤ 参加者URLを配布してスタート
```

---

## 4. イベント管理

イベントとは「スタンプラリーの開催期間と名称」をまとめた単位です。  
1つのイベントに複数のコースを紐づけます。

### イベントの追加

1. 「⚙️ イベント管理」をタップ
2. フォームに入力
   - **イベント名**：例）春日市ウォーキングラリー2026春
   - **説明**（任意）
   - **開始日時** / **終了日時**：期間を設定（省略可）
3. 「追加」ボタンをタップ
4. 「✓ イベントを追加しました」と表示されれば成功

### イベントの開始・停止

- イベント一覧の「開始」「停止」ボタンで切り替え
- **「開催中」のイベントのコースだけが参加者画面に表示されます**
- 期間（開始/終了日時）が設定されている場合、その期間外は自動で非表示になります

### コース・スポット構成の確認

イベント一覧の各イベントに、紐づくコースとスポットがツリー表示されます。

```
🎪 春日市ウォーキングラリー2026春 [開催中]
  🗺️ Aコース・北エリア
    1. 春日市役所
    2. 春日公園
    3. 春日駅
  🗺️ Bコース・南エリア
    1. 春日公園
    2. 奴国の丘
```

### 前回イベントからのコピー

イベントが2つ以上ある場合、コース・スポット構成を別のイベントにコピーできます。

1. 「コピー元イベント」を選択
2. 「コピー先イベント」を選択
3. 「コピーを実行」をタップ
4. QRコードは自動的に新規発行されます

> **用途例：** 「初詣ラリー2026」の構成をそのまま「春のラリー2026」にコピーしてスポットだけ変更する。

---

## 5. コース管理

コースはイベントに紐づけて作成します。

### コースの追加

1. 「🗺️ コース管理」をタップ
2. フォームに入力
   - **イベント** ※必須：どのイベントに紐づけるか選択
   - **コース名**：例）Aコース・北エリア
   - **説明**（任意）
   - **距離**（任意）：例）3.5（km）
   - **所要時間**（任意）：例）50（分）
3. 「追加」ボタンをタップ

> イベントを先に作成してからコースを追加してください。

---

## 6. スポット管理

スポットはコースに紐づけて作成します。

### スポットの追加

1. 「📍 スポット管理」をタップ
2. フォームに入力
   - **コース** ※必須：どのコースに紐づけるか選択
   - **スポット名**：例）春日市役所
   - **説明**（任意）
   - **住所**（任意）

3. **位置情報の入力（Google マップから）**

   **PCの場合：**
   - Google マップでスポットを右クリック
   - 表示される数字（例: `33.5298, 130.4688`）をクリックでコピー
   - 緑の入力欄に貼り付けて「入力」ボタンをタップ

   **スマホの場合：**
   - Google マップでスポットを長押し
   - 上部に表示される数字をタップしてコピー
   - 緑の入力欄に貼り付けて「入力」ボタンをタップ

4. 「追加」ボタンをタップ

> スポットを追加するとQRコードが自動生成されます。

---

## 7. QRコードの発行・設置

### QRコードの表示・印刷

1. 「🔲 QRコード」をタップ
2. 登録済みスポットのQRコードが一覧表示される
3. 右上の「印刷」ボタンをタップ → ブラウザの印刷ダイアログから印刷

### 設置の注意点

- ラミネート加工推奨（雨・汚れ対策）
- 参加者の目線の高さに設置
- 最小サイズ 5cm × 5cm 以上を推奨

---

## 8. 統計の確認

「📊 統計」では以下の情報を確認できます。

| 指標 | 内容 |
|------|------|
| 参加者数 | スタンプを1つ以上取得した人数 |
| スタンプ取得数 | 全参加者のスタンプ取得合計 |
| 完走者数 | 全スポットを制覇した参加者数 |
| 完走率 | 完走者数 ÷ 参加者数（%） |
| スポット別訪問数 | 各スポットの訪問回数ランキング |

---

## 9. 景品応募者の管理

1. 「🎁 景品応募者」をタップ
2. 応募者の名前・連絡先・応募日時が一覧表示

**ステータスの変更：**
- 「当選」ボタン → 緑色に変わる
- 「落選」ボタン → 赤色に変わる
- 「戻す」ボタン → 「未対応」に戻す

> 当選者への連絡は管理者が別途行ってください。自動通知機能はありません。

---

## 10. 参加者URLの案内

参加者が最初にアクセスするURLは、スーパー管理者から発行されます。

```
https://asoma.mago-t.com/?t=XXXXXXXX
```

このURLをチラシ・SNS・QRコードで配布してください。

> **重要：** 参加者はこのURLから入場しないとスタンプを取得できません。トップページ（`https://asoma.mago-t.com`）への直接アクセスでは参加できません。

---

## 11. よくある質問

**Q: イベントを追加したがコースが表示されない**  
A: コース管理でそのイベントにコースを追加してください。コースがないと参加者画面には何も表示されません。

**Q: 参加者画面にコースが表示されない**  
A: イベントが「開催中」になっているか確認してください。また、開始/終了日時が設定されている場合は期間内かどうか確認してください。

**Q: 参加者がスタンプを誤って取得してしまった**  
A: スーパー管理者に連絡してデータを削除してもらってください。

**Q: QRコードが読み取れないと言われた**  
A: 印刷サイズ（5cm×5cm以上）・汚れ・明るさを確認してください。

**Q: パスワードを変更したい**  
A: スーパー管理者に連絡してパスワードの変更を依頼してください。
