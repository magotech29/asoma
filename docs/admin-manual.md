# コミュニティ管理者マニュアル
## ぐるっとスタンプラリー 管理画面操作ガイド

---

## 1. ログイン

**URL：** `https://asoma.mago-t.com/admin/login`

| 項目 | 内容 |
|------|------|
| メールアドレス | スーパー管理者から発行されたメールアドレス |
| パスワード | スーパー管理者から発行されたパスワード |

ログイン後は **8時間** でセッションが切れます。再度ログインしてください。

---

## 2. 管理画面トップ

ログイン後、以下のメニューが表示されます。

| メニュー | 用途 |
|---------|------|
| 📊 統計 | 参加者数・完走率・スポット訪問数の確認 |
| 🎁 景品応募者 | 応募者一覧・当選/落選の管理 |
| 🗺️ コース管理 | ウォーキングコースの追加・編集・削除 |
| 📍 スポット管理 | チェックポイントの追加・編集・削除 |
| 🔲 QRコード | QRコードの表示・印刷 |
| ⚙️ イベント設定 | 開催期間・イベント情報の管理 |

---

## 3. イベント設定

スタンプラリーを開始する前に、まずイベントを設定します。

**手順：**
1. 「⚙️ イベント設定」をタップ
2. 「イベントを追加」フォームに入力
   - **イベント名**：例）春日市ウォーキングラリー2026
   - **説明**（任意）：イベントの概要
   - **開始日時**：スタンプラリー開始日時
   - **終了日時**：スタンプラリー終了日時
3. 「追加」ボタンをタップ
4. 作成されたイベントの「開始」ボタンをタップして有効化

> **注意：** イベントを「停止」にするとスタンプラリーは継続できますが、管理上の区切りとして使用できます。

---

## 4. コース管理

**コースの追加：**
1. 「🗺️ コース管理」をタップ
2. フォームに入力
   - **コース名**：例）春日市中心部コース
   - **説明**（任意）：コースの特徴
   - **距離**（任意）：例）3.5
   - **所要時間**（任意）：例）50（分）
3. 「追加」ボタンをタップ

**コースの編集・削除：**
- 一覧の「編集」または「削除」をタップ
- 削除するとそのコースに紐づくスポットも影響を受けるため注意

---

## 5. スポット管理

スポット（チェックポイント）はコースに紐づけて登録します。

**スポットの追加：**
1. 「📍 スポット管理」をタップ
2. フォームに入力
   - **コース**：どのコースに属するか選択
   - **スポット名**：例）春日市役所
   - **説明**（任意）：施設の説明
   - **住所**（任意）：例）福岡県春日市日の出町2-15
   - **緯度・経度**（任意）：マップ表示に使用
3. 「追加」ボタンをタップ

> **ポイント：** スポットを追加するとQRコードが自動生成されます。

---

## 6. QRコードの発行・設置

**QRコードの表示：**
1. 「🔲 QRコード」をタップ
2. 登録済みスポットのQRコードが一覧表示される

**QRコードの印刷：**
1. 右上の「印刷」ボタンをタップ
2. ブラウザの印刷ダイアログから印刷

**設置の注意点：**
- QRコードは雨・汚れから保護できるようラミネート加工を推奨
- 参加者が見つけやすい場所（目線の高さ）に設置
- 読み取りやすいよう最小サイズ5cm×5cm以上を推奨

---

## 7. 統計の確認

「📊 統計」では以下の情報を確認できます。

| 指標 | 内容 |
|------|------|
| 参加者数 | スタンプを1つ以上取得した人数 |
| スタンプ取得数 | 全参加者のスタンプ取得合計 |
| 完走者数 | 全スポットを制覇した参加者数 |
| 完走率 | 完走者数 ÷ 参加者数（%） |
| スポット別訪問数 | 各スポットの訪問回数ランキング |

---

## 8. 景品応募者の管理

**応募者一覧の確認：**
1. 「🎁 景品応募者」をタップ
2. 応募者の名前・連絡先・応募日時が一覧表示される

**当選/落選の設定：**
- 「当選」ボタン：当選者に設定（緑色に変わる）
- 「落選」ボタン：落選者に設定（赤色に変わる）
- 「戻す」ボタン：「未対応」に戻す

> **注意：** 当選者への連絡は管理者が別途行ってください。システムからの自動通知機能はありません。

---

## 9. 参加者向けURLの確認

参加者がスタンプラリーに参加するためのURLは、スーパー管理者から発行されます。

**形式：**
```
https://asoma.mago-t.com/?t=XXXXXXXX
```

このURLをチラシ・SNS・QRコードなどで参加者に配布してください。

---

## 10. よくある質問

**Q: 参加者がスタンプを誤って取得してしまった**  
A: Supabaseの管理画面（Table Editor → stamp_logs）から該当レコードを削除できます。スーパー管理者に依頼してください。

**Q: QRコードが読み取れないと言われた**  
A: 以下を確認してください。
- QRコードが汚れていないか
- 印刷サイズが十分か（5cm×5cm以上）
- 読み取り環境（暗すぎないか）

**Q: パスワードを忘れた**  
A: スーパー管理者に連絡してパスワードの再発行を依頼してください。
